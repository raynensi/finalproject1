# finalproject1
Titanic Survival Analysis 📌 Project Objective

The objective of this project is to analyze the Titanic dataset and build a machine learning model to predict whether a passenger survived the Titanic disaster based on features such as age, gender, passenger class, fare, and more.

📂 Dataset

The project uses the following files:

train.csv – Training dataset containing passenger details and survival status. test.csv – Test dataset used to predict passenger survival. gender_submission.csv – Sample submission file showing the expected prediction format. 🛠️ Technologies Used Python Pandas NumPy Matplotlib Seaborn Scikit-learn 📊 Project Workflow Load the Titanic dataset. Perform data cleaning and preprocessing. Conduct Exploratory Data Analysis (EDA). Handle missing values and encode categorical data. Train machine learning models. Evaluate model performance. Predict survival for the test dataset. 🎯 Expected Outcome

Develop a predictive model that accurately determines whether a passenger survived the Titanic disaster and gain insights into the factors that influenced survival.

📁Conclusion

The Titanic Survival Analysis project successfully explored the Titanic dataset and developed a machine learning model to predict passenger survival. Through data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, and model training, important factors influencing survival—such as gender, passenger class, age, and fare—were identified.

The project demonstrates how machine learning can be used to solve real-world classification problems by analyzing historical data and making accurate predictions. It also highlights the importance of data preprocessing and feature selection in improving model performance. Overall, this project provides valuable insights into the Titanic disaster while strengthening practical skills in data analysis, visualization, and predictive modeling using Python.
